# Factorial Program

## Описание
Данный проект представляет собой консольную программу на языке Python, 
которая запрашивает у пользователя ввод положительного целого числа и 
вычисляет его факториал.

Программа реализована в рамках учебной практики и демонстрирует основы 
обработки пользовательского ввода, обработки ошибок и использования 
стандартных библиотек языка Python.

## Функциональные возможности
- Запрос ввода положительного целого числа
- Вычисление факториала введённого числа
- Вывод результата на экран
- Обработка ошибок ввода (нечисловые данные, отрицательные числа)
- Использование стандартной библиотеки math для оптимизированного вычисления факториала

## Используемые технологии
- Язык программирования: Python
- Стандартная библиотека: math
- Система контроля версий: Git
- Хостинг репозитория: GitHub
